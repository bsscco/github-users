## github-users
깃헙 유저를 검색하고 유저를 좋아요, 검색할 수 있는 앱

## 사용 기술
- 싱글 액티비티
- MVI & AAC
- Kotlin(with Coroutine, Flow)
- Compose, Compose Navigation
- Paging Compose 
- 클린 아키텍쳐
- UseCase
- Repository
- 멀티 모듈
